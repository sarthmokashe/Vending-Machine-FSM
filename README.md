# 🥤 Vending Machine Controller using Verilog HDL

A simple **Finite State Machine (FSM)** based Vending Machine Controller designed using **Verilog HDL**.  
This project simulates a vending machine that accepts **5 Rs** and **10 Rs** coins, dispenses a product when the total amount reaches **15 Rs**, and returns change if required.

---

# 📌 Features

- Accepts:
  - 5 Rs coin
  - 10 Rs coin
- Dispenses product at 15 Rs
- Returns change when extra amount is inserted
- Designed using Mealy FSM
- Simulated using EDA Playground / GTKWave

---

# 🛠️ Technologies Used

- Verilog HDL
- FSM (Finite State Machine)
- EDA Playground
- GTKWave
